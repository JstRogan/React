# Morent - Car Rental Platform

Fullstack car rental application with React frontend and ASP.NET backend.

## Project Structure

```
morent-fullstack/
├── frontend/          # React + TypeScript + Vite
│   ├── src/
│   ├── package.json
│   └── ...
├── backend/           # ASP.NET Core API
│   ├── Controllers/
│   ├── Models/
│   ├── Program.cs
│   └── ...
├── docs/              # Shared documentation
├── .gitignore
└── README.md
```

## Getting Started

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
dotnet run
```
